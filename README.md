## Uso
- escribe el número de no terminales y dale enter
- Escribir cada producción y dar enter tras cada línea.
- después, el programa imprimirá si es LL(1), SLR(1), ambas o ninguna.
- Si toca analizar cadenas, introduce cada cadena y dale enter, para acabar el bloque, dar Enter en una linea que este vacía.
- Si dice “ambas”, responde T, B o Q
